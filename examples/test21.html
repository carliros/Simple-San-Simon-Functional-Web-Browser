Vista General

En este proyecto encontraras una descripcion de la implementacion de un Navegador Web con Haskell.

Abordamos este proyecto con el objetivo de experimentar las capacidades de Haskell commo Lenguaje de Programacion Funcional con evaluacion perezosa y de proposito general.

Podemos ver [10] que Haskell ha crecido y madurado lo necesario como para hacer aplicaciones no solo en el ambito academico, sino tambien comercial; Puedes ver una amplia cantidad de estas aplicaciones en el Hackage [11] de Haskell.

Asi en este proyecto estoy usando la libreria curl [12] para la comunicacion, entre los diferentes protocolos y el navegador web, la libreria uulib (Error Correcting Parser Combinators) [14] para parsear los documentos HTML y CSS, uuag (Attribute Grammar Preprocesor) [13] para procesar la informacion, y wxhaskell (Libreria Grafica) para renderizar los documentos HTML en la pantalla.
