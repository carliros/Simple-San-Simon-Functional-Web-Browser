Modelo Box de CSS

En este [ejemplo], todos los boxes generados por el web-browser tienen un area de margin, borde y padding.

Se tiene una hoja de estilos con 5 pixels de margin, 20 pixels de padding, y un borde de 3 pixeles, con colores red y black

Se puede ver estas propiedades en los elementos block (asi como en este mismo elemento), o tambien en este elemento (small) , o en este otro elemento (big).
