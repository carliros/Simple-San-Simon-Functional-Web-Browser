Simple San Simon Functional Web Browser by carliros

A Web Browser written with Haskell

3SWBrowser is a Web Browser written with a functional programming language, Haskell. It supports basic functionality of CSS and HTML.

Dependencies

uulib, uuagc, url, directory, tagsoup, containers, filepath, gd , curl, wxcore, wx

Install

make browser
